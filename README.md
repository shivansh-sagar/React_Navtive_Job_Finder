# JOB HUNTER

Job Hunter is a platform which fetches all the jobs posted over the Internet and makes Job finding very easy for the Developers seeking for Jobs.

- Search Jobs
- Popular Jobs
- NearBy Jobs
- Trending Jobs
- Cross Platform
- No Sign In Requied
- No user Data Collection
